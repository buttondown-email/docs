# Authentication

Clients should authenticate by passing the token key in the `Authorization` HTTP header, prepended with the string `Token `(note the trailing space!). For example:

<code>Authorization: Token 401f7ac837da42b97f613d789819ff93537bee6a</code>

You can find your API key in your [Programming page](https://buttondown.email/settings/programming).

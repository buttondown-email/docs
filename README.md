This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app), with Typescript and MDX (and Storybook!)

## Getting Started

```bash
npm install
npm run dev
```
